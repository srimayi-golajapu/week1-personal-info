# Week 1 - Personal Information Manager

## Project Overview
This is a Python program that stores and displays personal information.
It demonstrates the use of:
- Variables
- User input
- Input validation
- String formatting
- String methods
- Comments for code explanation

## Features
- Stores at least four pieces of personal information
- Accepts user input for at least two fields
- Validates basic user input
- Displays formatted output
- Uses string methods such as .strip(), .title(), .upper()

## How to Run

1. Make sure Python 3 is installed.
2. Open terminal in this folder.
3. Run:

   python personal_info.py
